# QiitaExtensions

QiitaのChrome拡張パック

## 内容

- AdventCalenderFeedSummary: 購読中のアドベントカレンダーをまとめて表示する
- AdventCalenderShowDetails: アドベントカレンダーのカレンダーにいいね数、ストック数、View数を表示する
